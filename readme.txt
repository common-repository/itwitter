== Description ==

en
Twitter plugin for Wordpress. A useful plugin for those who uses Twitter. Twitter announse for Wordpress and other.
Features:
ShortUrls: Allows you to create short links on the fly using the services bitly and tinyurl, also allows the use of a random service from the list.
Post 2 Twitter: Ability to automatically send a link and a description of his post at twitter. The message in twitter is based on a template created. For example "New blogpost: %title% %excerpt% %url%". The maximum length of the post in twitter is limited to 140 characters. If a tweet is too large, then skip the first description, then the title.
Replace: automatic replacement of words from the text of the type #tagname or @username in the posts and comments to links in twitter.
Last tweets: Adds the last tweets specified users of twitter. Allows you to select one of the four plases show tweets. For example in the sidebar or footer.


ru
Вордпресс плагин для твиттера. Полезный плагин для тех, кто пользуется твиттером. Анонсы в твиттер и другое.
Возможности:
Короткие ссылки: Позволяет создавать короткие ссылки на лету при помощи сервисов bitly и tinyurl, также позволяет использовать случайный сервис из списка.
Пост в твиттер: Возможность послать автоматом ссылку и описание своего поста в twitter. Сообщение в twitter получается на основе созданного шаблона. Например "New blogpost: %title% %excerpt% %url%". Максимальна длина поста в twitter ограничивается в 140 символов. Если tweet будет слишком большим, то сначала пропустится описания, а затем заголовок.
Замена: Автоматическая замена слов из текста вида #tagname or @username в постах и комментах на ссылки в twitter.
Последние твиты: Добавляет последние твиты указанных пользователей из twitter. Позволяет выбрать одно из четырех мест показа твитов. Например в sidebar или footer.

== Installation ==

Requirements:

Wordpress 2.3-2.8

PHP5
Widget compatible theme.

Copy the plugin files  in wp-content/plugins .

In Plugins activate iTwitter.

In settings-> iTwitter.
